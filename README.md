# 🚘 Automobiles Sales & Price Analytics | Power BI Project

## 📌 Overview
The Automobiles Sales & Price Analytics dashboard is an interactive Power BI project developed to analyze vehicle sales, pricing trends, mileage, fuel types, and transmission patterns. This dashboard helps generate valuable insights into automobile market trends through data visualization and business intelligence techniques.

---

## 🚀 Project Objectives
- Analyze automobile pricing trends
- Compare vehicle brands and mileage
- Study fuel type and transmission distribution
- Track average prices by model year
- Build an interactive analytics dashboard
- Practice real-world business intelligence reporting

---

## 🛠 Tools & Technologies Used
- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Visualization

---

## 📊 Dashboard Features
### ✔ KPIs
- Total Cars
- Average Price
- Average Engine Size
- Average Owner Count
- Average Mileage

### ✔ Visualizations
- Average Price by Brand
- Count of Cars by Fuel Type
- Average Price by Model Year
- Average Mileage by Brand
- Mileage vs Price by Brand
- Cars by Transmission

### ✔ Interactive Filters
- Brand Filter
- Fuel Type Filter
- Transmission Filter
- Model Year Filter

---

## 📈 Key Insights
- Automatic transmission vehicles had slightly higher counts.
- Average vehicle prices increased over recent model years.
- Tesla recorded the highest average mileage.
- Fuel type distribution remained balanced across categories.
- Different brands showed unique mileage and pricing patterns.
